问：对于7a+23b=1这个等式，找到一对整数对(a,b)，使其满足该等式且a为最小正整数，对于这个整数对(a,b)，能否确定7在mod 23下的逆？

答：a=23n+10，b=-7n-3，取n=0即可，即(a,b)=(10,-3)
